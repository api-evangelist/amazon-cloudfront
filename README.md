# Amazon CloudFront API

Amazon CloudFront is a content delivery network (CDN) service built for high performance, security, and developer convenience. CloudFront delivers data, videos, applications, and APIs to customers globally with low latency and high transfer speeds, all within a developer-friendly environment. CloudFront is integrated with AWS services including AWS Shield for DDoS mitigation, Amazon S3, Elastic Load Balancing, and Amazon EC2 as origins for your applications, and Lambda@Edge for running custom code closer to users.

## APIs

- **Amazon CloudFront API** - API for managing CloudFront distributions, cache behaviors, origin configurations, and edge functions for global content delivery.
